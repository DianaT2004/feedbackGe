# Claude AI API Key Setup Instructions
# ===================================

# 1. Create a .env file in your project root:
#    touch .env

# 2. Add these lines to your .env file:
#    CLAUDE_API_KEY=sk-ant-api03-your-actual-key-here
#    PORT=3001

# 3. Get your Claude API key from:
#    https://console.anthropic.com/

# 4. Replace 'sk-ant-api03-your-actual-key-here' with your real API key

# ⚠️  IMPORTANT SECURITY NOTES:
# - Never commit your .env file to git
# - Never share your API key publicly
# - Keep your .env file local only
# - Make sure .env is in your .gitignore

# Example .env file content:
CLAUDE_API_KEY=sk-ant-api03-ABC123-your-real-key-here
PORT=3001
